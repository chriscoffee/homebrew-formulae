# Homebrew Formulae

A [Homebrew tap] for personal installs of shit that I have opinions about

[Homebrew tap]: https://github.com/Homebrew/brew/blob/master/docs/Taps.md

## Usage

Tap this repository:

    brew tap chriscoffee/formulae

Install the packages it contains like any other Homebrew package:

    brew install ncspot-custom
